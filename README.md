# web-python-editor
Web-based Python3 editor + execution for Datacurve OA


### Features
- Containerized backend that uses execs to ensure safe execution of user-submitted python code
- Frontend written in TypeScript React as requested!
- Backend written with FastAPI as requested!


### How to run
- Make sure docker desktop is installed and running!
- To run the backend, simply run `make start` in the root directory
- To run the frontend:
    - cd into `/frontend`
    - run `npm i`
    - then run `npm run dev`
